# 安装

在命令行中输入执行以下命令:

# With NPM
$ npm install @zhengxs/vuepress-theme-library --save

# With Yarn
$ yarn add @zhengxs/vuepress-theme-library

# With PNPM
$ pnpm add @zhengxs/vuepress-theme-library
